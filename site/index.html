Everything you expect from a real ORM

Typed to the core, honest about SQL, and portable across four dialects.

◇

Typed, safe queries

Every query is checked at compile time. Write something that doesn't make sense, and it simply won't build — no runtime surprises.

⟳

Change tracking & saving

Track entities, mutate them, call saveChanges(). Ormit figures out what changed, saves it in the right order, and catches conflicting updates.

⇄

Relationships & loading

Load related data with include/thenInclude, without the performance trap of one giant joined query — or load it on demand instead.

△

Snapshot migrations

Migrations come from comparing your model to your last commit — never your live database — so they're predictable and easy to review in a PR.

◈

Plugins

Soft-delete, timestamps, and multi-tenancy ship built in — using the exact same plugin API you'd use to write your own.

⚡

Kysely engine, hidden

Every dialect produces clean, parameterized SQL. The engine underneath is fully swappable — it never leaks into your code.

Write LINQ, get SQL

Compose queries with a fluent, fully-typed API. If something can't translate to SQL, you'll know right away — not after it ships.

  • Operators — eq/in/between, startsWith/contains/like/toLower/toUpper.
  • Terminals — toList, first, single, count, sum/avg/min/max, toPage.
  • Escape hatch — parameterized fromSql tagged templates.
query.ts
const page = await db.orders
  .where(x => x.total.gte(100).and(x.status.neq('cancelled')))
  .orderByDescending(x => x.createdAt)
  .select(x => ({ id: x.id, total: x.total }))
  .toPage(1, 20);

Save like a Unit of Work

Track plain classes, mutate them, and persist everything in one call. Ormit figures out what changed, saves it in the right order, and enforces concurrency checks.

  • No duplicates — find() and query results return the same tracked instance.
  • Concurrency — a stale update throws ConcurrencyError, instead of silently overwriting.
  • Transactions — database.transaction() wraps multiple saves in one atomic unit.
